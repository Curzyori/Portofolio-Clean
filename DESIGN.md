# DESIGN.md

## Brand Voice & Inspiration: Vercel

A stark, minimalist developer-platform design using a black-and-white theme, subtle hairline borders, and geometric typography, accented by atmospheric mesh gradients.

---

## Colors

- **Canvas (Background)**: `#ffffff` (Light) / `#0a0a0a` (Dark)
- **Canvas Soft**: `#fafafa` (Light) / `#0d0d0d` (Dark)
- **Canvas Soft 2**: `#f5f5f5` (Light) / `#121212` (Dark)
- **Ink (Text)**: `#171717` (Light) / `#ededed` (Dark)
- **Body Text**: `#4d4d4d` (Light) / `#a0a0a0` (Dark)
- **Muted Text**: `#888888` (Light) / `#666666` (Dark)
- **Border / Hairline**: `#ebebeb` (Light) / `#222222` (Dark)
- **Primary Accent / CTA**: `#171717` (Light) / `#ffffff` (Dark)
- **Accent Link**: `#0070f3`

---

## Typography

- **Headers / Display**: Sans-serif (`Geist` / `Inter`), semibold (weight 600) with negative tracking (`-1.2px`).
- **Body**: Sans-serif (`Geist` / `Inter`), weight 400.
- **Labels / Code / Eyebrow**: Monospace (`Geist Mono` / `JetBrains Mono`), weight 400.

---

## Components & Layout

- **Pill Badges**: Pill-shaped badges (`rounded-full`) for technology tags.
- **Grid Layout**: Card-based grid with hairline borders, subtle hover scaling or border-color shift.
- **Atmospheric Gradient**: An ambient background glow mimicking Vercel's mesh gradient (violet to pink, cyan to blue) in the hero section.
